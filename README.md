# 🎓 AI-Powered Inclusive Learning Assistant

An intelligent web-based platform designed to support diverse learners by integrating multiple AI-powered features such as text summarization, speech-to-text, text-to-speech, and translation into a single system.

---

## 🚀 Features

* 🧠 **Text Summarization**
  Generate concise summaries from long text using NLP models.

* 🔊 **Text-to-Speech (TTS)**
  Convert text into audio for better accessibility.

* 🎙️ **Speech-to-Text (STT)**
  Convert spoken audio into text using AI (Whisper).

* 🌍 **Translation (English → Urdu)**
  Translate content to Urdu for better understanding.

* 📂 **File Upload & Text Extraction**
  Upload PDF, DOCX, or TXT files and extract text automatically.

* 📊 **Dashboard Analytics**
  Track usage statistics like summaries, uploads, audio generation, and translations.

---

## 🧠 Project Objective

This project aims to enhance **inclusive education** by providing tools that assist users with different learning needs, making content more accessible and easier to understand.

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **AI Models:**

  * Hugging Face Transformers (Summarization & Translation)
  * OpenAI Whisper (Speech-to-Text)
  * gTTS (Text-to-Speech)
* **Other Tools:** FFmpeg

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Uzairkahn/inclusive-learning-assistant.git
cd inclusive-learning-assistant
```

### 2. Create virtual environment

```bash
python -m venv fyp_env
fyp_env\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

---

## 💡 Usage

1. Open the application in your browser
2. Upload a file or enter text
3. Use features:

   * Summarize text
   * Translate to Urdu
   * Convert text to speech
   * Convert speech to text
4. View activity stats on dashboard
